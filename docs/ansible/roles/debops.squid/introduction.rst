Introduction
============

The ``debops.squid`` role can be used to install and manage Squid HTTP Proxy
Server.


Installation
~~~~~~~~~~~~

This role requires at least Ansible ``v2.4.0``. To install it, run:

.. code-block:: console

   user@host:~$ ansible-galaxy install debops.squid
